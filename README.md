## ✨ 为什么选择缘份居插件？

1. **单插件全能架构**：无需安装多个冗余扩展，一个核心插件即可进行清晰的功能分层，大幅降低 WordPress 系统认知负荷。
2. **零代码极速集成**：告别繁琐的底层 API 封装与调试，通过简易短代码（Shortcode，如 `[yuanfenju_bazi]`）即可在任意文章或页面渲染精美排盘表单。
3. **现代化前端 UI**：内置开箱即用的前端交互响应式表单，免去切图与 AJAX 联调烦恼，完美适配移动端与 PC 端。
4. **安全与性能双重保障**：前端代码完全开源透明，核心复杂天体历法运算运行于云端服务器，绝不拖慢您本地的 WordPress 性能。

## 📦 下载与快速安装指南

1. 📥 前往本项目的 **[Releases 页面](../../releases/latest)** 下载最新版本的 `yuanfenju-astrology-toolkit.zip` 安装包。（*注意：请勿直接下载 Source Code，务必下载打包好的插件 ZIP 包*）
2. ⚙️ 登录您的 WordPress 后台，进入 `插件` -> `安装插件` -> `上传插件`。
3. 🚀 选择刚才下载的 `.zip` 文件，点击安装并 **启用**。
4. 📖 详细后台短代码调用方法请参考：**[WordPress 测算插件安装与使用指南](https://doc.yuanfenju.com/appendix/wordpress.html)**。
