 缘份居 · 中国占星与命理测算工具包 (WordPress 插件)
====================================================

【插件简介】
这是一款强大且开箱即用的 WordPress 插件，可瞬间将您的网站转化为全功能的国学算命与排盘引擎。
无需编写任何代码，只需使用简单的短代码（Shortcode）即可在文章、页面或小工具中嵌入专业的测算表单。支持简体与繁体中文双语输出。

核心功能涵盖：八字排盘与精算、紫微斗数、奇门遁甲、老黄历择吉、每日运势及塔罗牌占卜等近百种专业命理工具。

【环境要求】
本插件采用底层原生架构开发，轻量无冗余。只要您的服务器环境满足 PHP 7.0 及以上，即可完美运行。为了获得最佳的安全性和后台体验，建议搭配 WordPress 5.2 或更高版本使用。

【运行模式说明】
本插件贴心地为您提供了两种运行模式，可在后台一键切换：

1. 沙盒模式（Sandbox - 默认）：
   完全离线的测试环境。使用内置的静态模拟数据展示排版效果，无需填写 API 密钥，不发起外部网络请求。

2. 实时模式（Live）：
   真实的生产环境。需配置有效的 API 密钥，表单数据将通过 HTTPS 加密传输至缘份居 API 进行实时的高精度推演。
   * 隐私安全：无论是本插件还是缘份居云端，均坚持“即算即毁”原则，不存储、不保留用户的出生时间等隐私数据。

【安装与使用步骤】
1. 上传安装：在 WordPress 后台管理面板中，导航至 [插件] -> [安装插件] -> [上传插件]，上传本 zip 压缩包并点击“启用”。
2. 后台配置：在 WordPress 左侧菜单栏找到新增的 [缘份居] 面板。
3. 获取密钥：如果您需要真实测算，请前往缘份居商户后台 (https://portal.yuanfenju.com) 免费注册并获取您的专属 API 密钥。
4. 填写密钥：将密钥填入插件设置页，并将运行环境切换为“Live (生产环境)”，保存更改。
5. 开启模块：在设置页勾选您需要开启的测算功能，复制对应的短代码（如 [yfj_bazi_paipan] ），粘贴到您的任何页面中即可发布上线。

【常见问题】
问：插件会拖慢我的网站速度吗？
答：完全不会。所有复杂的排盘逻辑（如紫微星曜计算、八字真太阳时转换）均在缘份居云端服务器瞬间完成，只将最终结果返回给您的网站显示，零本地性能损耗。

问：用户提交的数据会保存在我的 WordPress 数据库里吗？
答：完全不会。无论是插件本地还是缘份居云端 API，均遵循“即算即毁”原则。所有数据仅用于实时排盘计算，不会进行任何长期落盘存储，全面保护终端用户隐私。

问：前端页面支持多语言吗？
答：完全支持。输出内容已支持简体中文与繁体中文切换，插件内可跟随您的网站需求进行配置。
